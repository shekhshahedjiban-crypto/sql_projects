#  SQL Data Analytics & Database Systems

This repository contains my professional projects focusing on database design, relational modeling, and advanced SQL querying. Each project demonstrates the ability to structure data effectively and extract actionable insights.

---

##  Projects Overview

### 1. E-commerce Sales Management System
A comprehensive database design for an online retail business.
*   **Key Features:** Automated revenue tracking, advanced category analysis, and complex customer invoice generation.
*   **SQL Techniques:** `JOIN`s, `GROUP BY`, `CASE` statements, and Aggregation.

### 2. HR & Employee Performance Analysis System
An analytical HR database designed for performance tracking and salary benchmarking.
*   **Key Features:** Departmental salary ranking, tenure analysis, and salary gap evaluation.
*   **SQL Techniques:** Window Functions (`ROW_NUMBER`, `RANK`, `DENSE_RANK`), `LAG()` & `LEAD()` functions, and CTEs.

### 3. Library Management System
A relational database solution for managing library resources and member transactions.
*   **Key Features:** Automated borrow/return tracking, member data management, and resource availability status.
*   **SQL Techniques:** Database Normalization, Foreign Key Constraints, and Relational Joins.
